# 🛍️ Retail Analytics: Predicting Customer Purchase Behavior

## 📌 Project Overview
This project analyzes retail transaction and marketing campaign data to uncover customer behavior, purchasing patterns, and campaign engagement trends. Using Excel, SQL, Python, and Tableau, the project applies data analytics techniques to predict customer responses to marketing campaigns and optimize promotional strategies to improve business performance and maximize ROI.

---

## 🎯 Business Objective
Retail businesses generate massive amounts of customer and transaction data, but extracting meaningful insights remains a challenge. This project focuses on transforming raw retail data into actionable insights that support:

- Customer segmentation
- Campaign response prediction
- Purchasing behavior analysis
- Marketing optimization
- Data-driven decision-making

---

## 📂 Dataset Information

### Datasets Used
- `marketing_campaign.csv`
- `marketing_campaign_Metadata.csv`

### Dataset Includes
- Customer demographics
- Income and education levels
- Marital status and household composition
- Product purchase history
- Campaign interaction and response data
